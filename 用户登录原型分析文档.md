## 🔧 功能特性

### 核心功能

**1. 用户登录**
- 支持用户名/邮箱/手机号登录
- 密码输入支持可见性切换

**3. 错误处理**
- 字段级错误提示 (账号下方、密码下方)

- 输入框焦点效果 (紫色边框 + 阴影)
- 页面加载自动聚焦到账号输入框
- 键盘快捷键 (Escape清空表单)

**5. 加载状态**
- 登录按钮loading状态 (菊花转)
- 加载期间禁用输入框
- 按钮文字淡出,spinner淡入

## 💻 技术实现

### HTML结构

```
登录容器 (login-container)
├── 登录头部 (login-header)
│   ├── h1: 系统登录
│   └── p: 欢迎回来...
│
├── 登录表单区 (login-form)
│   ├── h2: 账号登录
│   ├── 全局错误提示 (global-error)
│   └── form#loginForm
│       ├── 账号输入组
│       ├── 密码输入组 (带可见性切换)
│       └── 登录按钮
│
└── 登录底部 (login-footer)
    └── p: 版权信息
```

### CSS技术要点

**1. 渐变背景**
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

**2. 卡片效果**
- border-radius: 12px (圆角)
- box-shadow: 0 10px 40px rgba(0,0,0,0.1) (阴影)
- 动画: fadeInUp 0.5s (淡入上升)

**3. 输入框交互**
- 焦点时: border-color变色 + box-shadow光晕
- 错误时: 红色边框 + 红色背景
- transition: all 0.3s ease (平滑过渡)

**4. 按钮状态**
- 默认: 渐变背景
- hover: 上移2px + 阴影增强
- active: 归位
- disabled: 灰色背景 + 不可点击
- loading: spinner动画显示

**5. 动画效果**
- fadeInUp: 页面加载淡入上升
- shake: 错误抖动
- spin: 加载菊花转
- slideInRight: 成功提示滑入

**6. 响应式设计**
```css
@media (max-width: 480px) {
    .login-container {
        padding: 32px 24px;
    }
}
```

### JavaScript逻辑

**1. DOM元素获取**
- 表单、输入框、按钮、错误提示元素

**2. 事件监听**
- load: 页面加载自动聚焦
- click: 密码可见性切换
- input: 输入时清除错误
- blur: 失焦时验证
- submit: 表单提交处理
- keydown: 键盘快捷键

**3. 验证函数**
```javascript
validateUsername()  // 验证账号长度 >= 3
validatePassword()  // 验证密码长度 >= 6
```

**4. 错误处理**
```javascript
showError(field, message)    // 显示字段错误
clearError(field)            // 清除字段错误
showGlobalError(message)     // 显示全局错误
hideGlobalError()            // 隐藏全局错误
```

**5. 登录处理**
```javascript
handleLogin(username, password)
├── 设置loading状态
├── 禁用表单
├── 模拟API请求 (setTimeout 1500ms)
├── 验证账号密码 (admin/admin123)
├── 成功: 显示Toast + 模拟跳转
└── 失败: 显示错误 + 清空密码
```

**6. 成功提示**
```javascript
showSuccessToast(message)
├── 创建Toast元素
├── 添加到body
├── 3秒后移除
```
